Generative Art

The designs below were created using L-Systems. An L-System starts with a short string of characters and with each generation, characters are replaced based on a pre-determined set of rules. Then, the string is drawn to screen using turtle graphics, where each character represents a command.

The rules that generated these designs were made with intention to perserve symetry. In making them, I drew inspiration from famous fractals like the Levy Tapestry and the Sierpi≈Ñski Carpet.

This next selection of designs were made with a technique inspired by branching patterns. The L-System starts with a few squares and circles and at each generation the shapes get replaced by one or more paths that end with a square or a circle. The result is a intricate geometric web.

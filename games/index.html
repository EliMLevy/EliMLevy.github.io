Kill Gil(bert)

In this game the player is tasked with clearing a room full of enemies called 'Gilbert' (I hoped that humanizing them would add an extra layer of difficulty to the game). Killing all the Gilberts in a room allows the player to advance to the next room with stronger, faster, and unique Gilberts. Killing the Gilberts will award the player with coins that he/she can use to buy upgrades in the shop. Use those upgrades to get father!
Controls: WASD to move, mouse to aim, click to shoot, 'R' to reload.
I had a lot of fun making Kill Gil(bert). I began working on it as I learnt object oriented programming. This project helped me learn how to design particle systems and use class inheritence. Later, I added the leaderboard system which was my introduction to HTTPS requests. I used the githb gists API to store and retrieve data.

Kill Gil Arena Brawl

This game relies on similar mechanics to the original Kill Gil(bert) however this time the enemies are other players! Players who are playing at the same time will be able to fight each other to the death! Move around the arena launching fireballs at your oponent. Be careful though, the obstacles deflect the fireballs and its very easy to end up hitting yourself.
Controls: WASD to move, mouse to aim, click to shoot.
During my winter break of freshman year I set out to learn Node.js. Learning server-side programming was incredibly fun and very challenging. This project is made using express, socket.io and is hosted on glitch.me.

Boop the Stroop

Click the color that is written on the screen before the timer runs out to stay alive. When you run out of time you lose! The name is based of the Stroop effect (read more about it here)
This is the simplest game on my website. I built it as a excuse to learn a better way to create a leaderboard than with github gists. This project uses Firebase Realtime Databse (check it out here) to store and retrieve data. Their documentation was super clear and I found it to be rather intuitive and super effective.

Skiddadle!

Try to make it to the goal without gettting eaten by Joe (the red face that tries eating you). Try picking up gems along the way but be warned: I didn't make a shop for this game so they are quite useless. Controls: WASD to move.
This is the second version of Skiddadle (you can play version 1 here). The first version was the first game I ever made in JavaScript. So, while it isnt much, it has a special place in my heart. In Version two I wanted to take it a step further and experiment with a physics engine. So I learnt how to use Matter.js (check it out here), a simple 2D JavaScript physics engine. Figuring out how to use Matter.js was frusterating and I was never quite satisfied the gameplay but I'm glad that I got to experience using a physics libary.

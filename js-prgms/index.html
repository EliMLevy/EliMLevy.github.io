P5.JS Programs

This is a diffusion-limited aggregation simulation. Read more about it here. Use the buttons to start with different seeds. Control the speed of the simulation with the speed slider. The population slider allows the simulation to run more than one particle at a time. Making the speed and population too high can result in poor perfomance or even causing the program to crash.

Here is a sudoku solving algorithm that uses a depth first technique. There are ten puzzles loaded in that can be accessed with the 'Random puzzle' or you can edit the board by clicking on the desired square and using 1-9 and 'backspace' to clear.

This is a maze generator that uses a depth first search algorithm. More information about it can be found here. The mazes that it generates aren't particularly difficult but it's pleasant to watch it go!
